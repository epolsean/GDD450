*** Strategix: Etherian Conflict ***



Instructions:

Use mouse for almost all controls.
Having the mouse near the edges of the screen (or using WASD) will move the camera.
Scroll wheel on the mouse will allow zooming in and out.
To select a hex, left click the mouse.

During battles a simulation will run that will decide the winner.
The goal of the game currently is to eliminate all of the enemy pieces.
There will be other modes coming soon(Domination and Control Points).
If done the person with piece left wins.
